# Test Nutrialus

Desarollo del test para Nutrialus con React Native con Expo CLI

## Instalar

 1. Clona este repo
 2. Abre la carpeta `cd test-nutrialus`
 3. Instala las dependencias con `yarn` รณ `npm install`
 4. Ejecuta Expo con `Yarn start` รณ `npm start`
